# Biz2Optima - Tax & Business Consultancy Website

Built for a real client **Biz2Optima** (operating in AU, PK, GB),                     
a trusted tax & business consultancy serving Pakistan, Australia, and the UK.

**Live Demo:** [Biz2Optima.vercel.app](https://taxation-olive.vercel.app)

---

## Tech Stack

**Frontend:** React, JavaScript, CSS  
**Backend:** Node.js, Express  
**Deployment:** Vercel (client) · Render (server)

---

## Features

- Multi-region support (AU · PK · GB)
- Fully responsive single-page layout
- Working contact form with backend integration
- Services, FAQ, Insights sections
- WhatsApp chat button integration

---

## Project Structure

├── client/   # React frontend → deployed on Vercel

└── server/   # Express backend → deployed on Render

---

## Run Locally

**Frontend:**

cd client

npm install

npm run dev

**Backend:**

cd server

npm install

npm start
